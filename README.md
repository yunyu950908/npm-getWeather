# npm-getWeather
one light npm application

## 说明
一个命令行查询天气的轻应用

## 使用方法
- 安装
```bash
npm install -g weather-liyu
```
- 查询
```bash
//直接查询当前IP所在城市的天气
weather
```
![](https://ws1.sinaimg.cn/large/889b2f7fgy1fikyhx1oy8j207m03fwee.jpg)

```bash
//查询指定城市的天气
weather [CurrentCity]
```
